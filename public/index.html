Como Jogar

Use as teclas WASD para mover seu personagem.

Clique com o mouse para atirar. Cada jogador pode dar 7 tiros antes de precisar recarregar.

Pressione a tecla R para recarregar.

Pressione a tecla G para lançar uma granada.
